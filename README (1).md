# ⌨ Neki Macro v.1

โปรแกรมมาโครคีย์บอร์ด (Keyboard Macro) พร้อม UI แบบ Sidebar ทันสมัย ตั้งค่าปุ่ม Trigger เอง, กำหนดลำดับการกดปุ่ม/พิมพ์ข้อความ, ตั้งค่าดีเลย์ (delay) และมีศูนย์ **Backup & Restore** ที่อธิบายข้อดี/ข้อเสียของทุกการกระทำอย่างชัดเจนก่อนดำเนินการ

**Written By Neki v.1**

## ✨ ฟีเจอร์หลัก

- ✅ **Sidebar Navigation** แยกหน้า "มาโคร" และ "Backup & Restore" ชัดเจน
- ✅ ตั้งค่า **Key Binding** ได้เอง (ปุ่ม Trigger เดี่ยว หรือ combo เช่น `ctrl+shift+m`)
- ✅ **Custom ปุ่ม/ขั้นตอน** ได้ไม่จำกัด — กดปุ่ม, พิมพ์ข้อความ, หรือหน่วงเวลา
- ✅ ตั้งค่า **Delay** เป็นมิลลิวินาที ได้ทั้งระดับ step และแบบ delay step แยก
- ✅ ตั้ง **จำนวนรอบที่ทำซ้ำ** ของมาโคร (หรือทำซ้ำไม่จำกัดจนกว่าจะหยุด engine)
- ✅ เปิด/ปิดมาโครแต่ละตัวได้ (checkbox enable/disable)
- ✅ **Export / Import Backup** เป็นไฟล์ .json พร้อมคำอธิบายข้อดี-ข้อเสียก่อนทำทุกครั้ง
- ✅ **ลากไฟล์ Backup วางเพื่อกู้คืน** ได้ทันที (drag & drop)
- ✅ **รีเซ็ตเป็นค่าเริ่มต้น** พร้อมสร้าง Safety Backup อัตโนมัติก่อนล้างข้อมูลเสมอ
- ✅ ทุกการกระทำที่มีผลกระทบ (ลบ/กู้คืน/รีเซ็ต) มี**กล่องยืนยันแบบละเอียด** บอกทั้งข้อดีและข้อเสียก่อนกดยืนยันจริง
- ✅ มี Log แสดงกิจกรรมแบบเรียลไทม์
- ✅ UI ธีมมืดทันสมัยด้วย CustomTkinter

## 🔧 การติดตั้ง

ต้องมี Python 3.9+ ติดตั้งอยู่ในเครื่องก่อน จากนั้นรันคำสั่ง:

```bash
pip install -r requirements.txt
```

หรือติดตั้งทีละตัว:

```bash
pip install customtkinter keyboard tkinterdnd2
```

> ⚠ **สำคัญ (Windows):** ควรรันโปรแกรมแบบ "Run as Administrator" เพื่อให้ไลบรารี `keyboard`
> สามารถดักจับปุ่มคีย์บอร์ด (global hotkey) ได้ในทุกโปรแกรม
>
> ⚠ **Linux:** ต้องรันด้วย `sudo` เนื่องจาก `keyboard` ต้องการสิทธิ์เข้าถึงอุปกรณ์อินพุตโดยตรง
>
> ℹ️ **tkinterdnd2:** ใช้สำหรับฟีเจอร์ลากไฟล์ Backup วาง หากไม่ได้ติดตั้ง โปรแกรมยังใช้งานได้ปกติ
> เพียงแต่ต้องใช้ปุ่ม "เลือกไฟล์" แทนการลากวาง

## 📖 หน้าจอที่ 1: มาโคร (หน้าหลัก)

1. กดปุ่ม **"+ มาโครใหม่"** เพื่อสร้างมาโคร
2. ตั้งชื่อมาโคร แล้วกด **"🎯 จับปุ่ม"** ที่ช่อง Trigger เพื่อกดปุ่มจริงที่ต้องการใช้เป็นตัวเริ่มมาโคร
3. ตั้งจำนวนรอบที่ต้องการทำซ้ำ (ใส่ `0` = ทำซ้ำไปเรื่อยๆ จนกว่าจะกด "หยุดทำงาน")
4. กด **"+ เพิ่ม Step"** เพื่อเพิ่มขั้นตอน โดยเลือกประเภท:
   - **key** — กดปุ่ม (กด "🎯 จับปุ่ม" เพื่อระบุปุ่มจริง หรือพิมพ์เอง เช่น `ctrl+c`)
   - **text** — พิมพ์ข้อความอัตโนมัติ
   - **delay** — หน่วงเวลาก่อนไปขั้นตอนถัดไป (หน่วยเป็น ms)
5. ใช้ปุ่ม **⬆ ⬇** เพื่อจัดลำดับ step, **แก้ไข/ลบ** เพื่อจัดการ step ที่มีอยู่
6. กด **"บันทึกมาโคร"** เพื่อบันทึก
7. กลับมาหน้าหลัก กด **"▶ เริ่มทำงาน"** เพื่อเปิดใช้งานมาโครทั้งหมดที่ enable ไว้
8. กดปุ่ม Trigger ที่ตั้งไว้ ณ ที่ใดก็ได้บนเครื่อง มาโครจะถูกรันทันที
9. กด **"⏸ หยุดทำงาน"** เพื่อปิด engine

## 💾 หน้าจอที่ 2: Backup & Restore Center

หน้านี้ถูกออกแบบมาให้อธิบายผลกระทบของทุกการกระทำอย่างชัดเจน ก่อนที่คุณจะกดยืนยันจริง:

### 📤 Export Backup
กดปุ่ม "Export Backup ตอนนี้" เพื่อบันทึกมาโครทั้งหมดที่มีอยู่เป็นไฟล์ `.json` เก็บไว้ในที่ที่เลือก
- **ข้อดี:** ทำได้ไม่จำกัดจำนวนครั้ง ไม่กระทบมาโครที่ใช้งานอยู่
- **ข้อควรระวัง:** ต้องมากด Export เองทุกครั้งที่แก้ไข ระบบไม่ backup อัตโนมัติให้

### 📥 Restore จากไฟล์ Backup
- **ลากไฟล์ .json มาวางในกล่อง** ที่ระบุไว้ในหน้า (ถ้าเครื่องรองรับ) หรือกด **"เลือกไฟล์ Backup..."**
- ระบบจะเปิดกล่องยืนยันบอกจำนวนมาโครในไฟล์ พร้อมข้อดี-ข้อเสียก่อนกู้คืนจริง
- **ข้อดี:** กู้คืนมาโครทั้งหมดกลับมาได้ในไม่กี่วินาที
- **ข้อควรระวัง:** มาโครปัจจุบันทั้งหมดจะถูกแทนที่ทันที ถ้ายังไม่เคย backup ไว้จะหายถาวร

### 🔄 รีเซ็ตเป็นค่าเริ่มต้น (Factory Reset)
ล้างมาโครทั้งหมด กลับไปเป็นสถานะว่างเปล่าเหมือนเปิดโปรแกรมครั้งแรก
- ระบบจะ **สร้าง Safety Backup ให้อัตโนมัติก่อนล้างข้อมูลทุกครั้ง** (เก็บไว้ในโฟลเดอร์ `backups/`)
- **ข้อดี:** เริ่มต้นใหม่แบบสะอาด และมี safety backup กู้คืนได้เสมอ
- **ข้อควรระวัง:** มาโครทั้งหมดในหน้าจอหลักจะหายไปทันทีหลังยืนยัน

## 🌐 Build เป็น .exe ผ่าน GitHub Actions (ไม่ต้องมีเครื่อง Windows)

1. สร้าง repository ใหม่บน [github.com](https://github.com)
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo (รวมโฟลเดอร์ `.github/` ด้วย — สำคัญมาก)
3. เข้าไปที่แท็บ **Actions** ของ repo บนเว็บ GitHub
4. workflow **"Build Neki Macro EXE"** จะรันอัตโนมัติ (หรือกด **Run workflow** เพื่อรันเอง)
5. รอสัก 2-4 นาที ให้สถานะขึ้นเครื่องหมายถูกสีเขียว ✅
6. เลื่อนลงล่างสุดของหน้า run ไปที่ **Artifacts** → ดาวน์โหลด **NekiMacro-Windows-EXE.zip**
7. แตกไฟล์ ได้ `NekiMacro.exe` พร้อมใช้งานบน Windows ทันที

## 💡 ตัวอย่างการตั้งค่ามาโคร

**ตัวอย่าง 1: มาโครกด Copy-Paste อัตโนมัติ**
- Trigger: `f6`
- Steps: `ctrl+c` (delay 100ms) → delay `300` → `ctrl+v`
- Repeat: `1`

**ตัวอย่าง 2: พิมพ์ข้อความซ้ำๆ ทุก 1 วินาที จนกว่าจะหยุด**
- Trigger: `f9`
- Steps: text `สวัสดีครับ` (delay 1000ms)
- Repeat: `0` (ไม่จำกัด)
